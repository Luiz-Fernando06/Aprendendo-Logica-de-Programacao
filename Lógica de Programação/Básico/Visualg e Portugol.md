# Comando de saída 

> 	Escreva("Qualquer coisa") 
> 	Escreval("Qualquer coisa")



- **O uso das Aspas** nós algoritmos são muito importantes, pois elas são consideradas uma mensagem, uma cadeia de caracteres 

- **Escreval** é apenas do visualg para pular linha, no Portugol usamos o **\n**

---

# Atividade: Olá mundo

### ==**Visualg**== 

> 	Algoritmo "OlaMundo"
> 	    Var
> 	    
> 	Inicio
> 	    Escreval("Olá, Mundo!")
> 	    Escreva("Finalmente me livrei da maldição!")
> 	    
> 	FimAlgoritmo



### ==**Portugol**==

> 	programa
> 	{
> 		funcao inicio ()
> 		{
> 		escreva("Ola mundo! \n")	
> 		escreva("Me livrei da maldição")
> 			
> 		}
> 	}
